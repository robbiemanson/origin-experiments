Use to apply `letter-spacing` values with utility classes.

### Values
* `0`
* `inherit`
* `normal`

### Examples
{% example html %}
<div class="u-letter-spacing--0">
  This will overwrite any inherited `letter-spacing` values with `0`
</div>
{% endexample %}
